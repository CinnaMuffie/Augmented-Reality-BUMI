# Augmented-Reality-BUMI
Augmented Reality Bumi adalah proyek eksplorasi visual interaktif yang memanfaatkan teknologi AR untuk menampilkan model 3D bumi secara real-time. Proyek ini bertujuan memberikan pengalaman belajar dan eksplorasi yang lebih imersif tentang planet kita, baik untuk edukasi maupun eksperimen kreatif.
